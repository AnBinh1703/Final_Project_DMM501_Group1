# Real-Time Fraud Detection System for Financial Transactions

> University final project (AI in Production / ML Systems Engineering)

This repository will contain an end-to-end, production-style ML system for real-time fraud detection on tabular transaction data, including:

- Training pipeline + experiment tracking (MLflow)
- Real-time inference API (FastAPI)
- Containerized deployment (Docker Compose)
- Monitoring (Prometheus + Grafana)
- Testing (pytest) + CI (GitHub Actions)

See ARCHITECTURE.md for system design and CONTRIBUTING.md for developer workflow.
